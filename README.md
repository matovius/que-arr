# QuAr (que-arr)

My very own QR code generator app. It's as simple as it gets, which is why there's more CSS code than Svelte/HTML.

I wanted to more easily share code from my computer to my phone, so I made this. It was very fun and such quick turn around.

This app doesn't connect to a backend, so it's completely client-only. You may find a deployed version [here](https://qr.matovius.dev). Explore around if you need, and thanks for visiting, take care!
